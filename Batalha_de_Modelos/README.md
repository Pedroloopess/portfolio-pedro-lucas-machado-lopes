# 🚀 Engenharia de Prompt e Aplicações em IA

## Batalha de Modelos & Engenharia de Prompt (XML)

> Atividade desenvolvida com foco na comparação entre diferentes modelos de Inteligência Artificial utilizando um prompt estruturado em XML para geração de páginas HTML Single Page com CSS integrado.

---

# 📖 Descrição

Este projeto teve como objetivo criar um **Prompt Estruturado em XML** capaz de gerar uma página HTML5 moderna e responsiva utilizando CSS interno.

O mesmo prompt foi testado em diferentes modelos de IA generativa, permitindo analisar:

* Precisão da interpretação do XML
* Qualidade do HTML gerado
* Criatividade visual
* Quantidade de erros
* Consumo de tokens
* Eficiência geral de cada modelo

---

# 🎯 Objetivo

Avaliar o desempenho de diferentes modelos de IA na interpretação de prompts estruturados, observando:

* Capacidade de compreensão semântica
* Qualidade da geração de código
* Organização visual da interface
* Eficiência de tokens
* Robustez técnica
