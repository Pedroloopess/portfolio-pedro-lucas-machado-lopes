## 📝 Descrição do Projeto

Este projeto explora a união entre criatividade humana e Inteligência Artificial Generativa aplicada à música.

A proposta foi criar um EP autoral com 5 faixas inéditas utilizando o **Gemini Gems** para análise lírica e geração de letras, e o **Suno AI** para produção musical, vocais e instrumentais.

O projeto utiliza conceitos de **RAG (Retrieval-Augmented Generation)**, engenharia de prompt e composição generativa para criar músicas com estética emocional, moderna e atmosférica.

---

## 🎯 Objetivo

Desenvolver um projeto musical completo utilizando Inteligência Artificial, demonstrando como ferramentas generativas podem auxiliar no processo criativo, desde a análise de referências até a criação de faixas finalizadas.

## 🚀 Assistente Personalizado

O assistente utilizado no Gemini Gems foi configurado para analisar letras, identificar padrões emocionais e gerar composições originais inspiradas apenas na essência estética das referências.

Arquivo utilizado:

```bash
prompts-gemini.txt
```

O assistente foi orientado a identificar:

- gênero musical;
- atmosfera;
- BPM aproximado;
- estrutura da música;
- estilo vocal;
- padrões de repetição;
- temas principais;
- linguagem utilizada;
- estética emocional.

---

## 🎧 Prompt Base para Suno AI

O prompt utilizado no Suno AI foi escrito em inglês, com foco em uma sonoridade de trap melódico brasileiro, cloud rap e dark trap.

Arquivo utilizado:

```bash
prompts-suno.txt
```

A estética sonora definida inclui:

- 808 grave e distorcido;
- synths atmosféricos;
- vocais masculinos com autotune;
- pads sonhadores;
- guitarras suaves;
- texturas cinematográficas;
- clima noturno e melancólico.

## 📄 Processo de Criação

### 1. Curadoria

Foram selecionadas referências musicais e estéticas para orientar o assistente personalizado no Gemini Gems.

### 2. Prompt Engineering

Foram criadas instruções específicas para que o assistente analisasse características líricas e musicais sem copiar versos, melodias ou estruturas completas.

### 3. Geração das Letras

O Gemini Gems foi utilizado para criar letras originais com linguagem emocional, simples e urbana.

### 4. Produção Musical

As letras e os prompts foram levados ao Suno AI para gerar as músicas completas, com vocais, batidas e instrumentais.

### 5. Organização no GitHub

Os arquivos foram organizados em uma estrutura simples, com áudios, prompts e documentação em README.md.
