## 📝 Descrição do Projeto

Este projeto explora a intersecção entre criatividade humana e Inteligência Artificial. Utilizando técnicas de RAG (Retrieval-Augmented Generation) através do Gemini Gems, foi criado um assistente especializado na análise lírica e estética musical do artista SOTAM.

O objetivo final foi a produção de um EP com 5 faixas inéditas, cujas letras foram geradas através de IA e a produção musical realizada utilizando o Suno AI.

---

# 🎯 Objetivo

Desenvolver um pipeline criativo utilizando IA generativa para:

- análise lírica;
- engenharia de prompts;
- geração de letras originais;
- criação de músicas completas;
- produção de um EP inspirado na estética do trap melódico brasileiro.

## 📄 Processo de Criação

### 1. Curadoria Musical
Seleção de letras e referências do artista SOTAM para identificação de padrões emocionais, estéticos e estruturais.

### 2. Prompt Engineering
Criação de prompts personalizados para o Gemini Gems com foco em:

- métrica;
- vocabulário;
- estética emocional;
- BPM;
- atmosfera musical;
- composição narrativa.

### 3. Produção Musical
As letras geradas foram convertidas em músicas através do Suno AI utilizando parâmetros como:

- BPM entre 120-140;
- synths atmosféricos;
- 808 graves;
- guitarras melódicas;
- reverbs cinematográficos;
- vocais autotunados emocionais.
