# Instruções para Entrega

Este projeto já está pronto para ser enviado ao GitHub.

## Como usar

1. Extraia o arquivo `.zip`.
2. Abra a pasta `projeto-musica-ia`.
3. Confira se os arquivos estão organizados assim:

```bash
projeto-musica-ia/
├── audio/
├── docs/
├── prompts-gemini.txt
├── prompts-suno.txt
└── README.md
```

4. Crie um repositório no GitHub chamado `projeto-musica-ia`.
5. Envie todos os arquivos da pasta para o repositório.
6. O README.md já está pronto para aparecer automaticamente na página inicial do GitHub.

## Observação

Caso o professor solicite o link do Gemini Gems, substitua no README a parte do assistente personalizado pelo link real do seu Gems.
